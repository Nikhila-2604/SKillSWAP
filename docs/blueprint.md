# **App Name**: SkillBarter

## Core Features:

- User Authentication: Authenticate users via Firebase Authentication, restricted to .edu domains.
- Real-time Task Feed: Display a real-time feed of 'Help Wanted' posts on the dashboard using Firestore.
- Post a Task: Enable users to submit tasks with category, description, duration, and location details.
- Skill Marketplace: Implement a searchable directory of users and their skills.
- Time-Credit Transactions: Manage time-credit balances and transaction history in a visual ledger, powered by Firestore.
- Balance Validation: Ensure users can't request tasks exceeding their timeBalance.
- Automated Credit Transfers: Automatically transfer credits when tasks are marked 'Completed' by both users.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) evoking trust and community.
- Background color: Light cyan (#E0FFFF), a subtle variant of sky blue to keep things light
- Accent color: Soft green (#98FB98) for CTAs, symbolizing growth and positive transactions.
- Body and headline font: 'PT Sans' for a clean, modern academic look.
- Use simple, academic-themed icons (e.g., books, clocks, gears) for categories and actions.
- Employ a clean, grid-based layout with clear sections for tasks, profiles, and balances.
- Use subtle transitions and animations for task updates and balance changes.
